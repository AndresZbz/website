# Starting using Vim

Since I started developing, I used ton of text editors and IDE's such as Atom, Sublime Text, Notepad++ and the one I'm using currently VS Code.

I knew about Vim since the start, and tried it a few times, but never got used to it, until now, as a way to "increase" my workflow by only using the keyboard. Learning using they main shortcuts was not difficult, learning how to open files, close files, the famoues ":q" to exit vim, etc. With `vim-plug` I also installed
a few packages, like nerd-tree and the gruvbox theme.

Right now I'm using vim for smaller project such as one file project or less than 10, but for now I wil still using VSCode as my main text editor until I feel I'm good enough to switch fully to Vim.
